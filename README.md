# ICEBERG — сайт группы компаний (прототип)

Одностраничный прототип корпоративного сайта группы компаний «Айсберг».
Полностью самодостаточный `index.html` (шрифты, логотип и графика встроены,
внешних зависимостей нет). Можно открыть локально или отдать под хостинг /
GitHub Pages.

Разделы: хиро, «Наши компании» (scrollytelling), группа в цифрах,
клиенты (бегущая строка), «Для кого», контакты с формой заявки.
