# ICEBERG — сайт группы компаний (прототипы)

Прототипы корпоративного сайта группы компаний «Айсберг».
Все версии закрыты паролем и скрыты от поисковой индексации (`noindex`).

## Версии

| | Ссылка | Описание |
|---|---|---|
| **v3** | [`/v3/`](https://timurthejohn-droid.github.io/iceberg-website/v3/) | Актуальная. Первый экран — техническая ткань, разворачивается по скроллу. Заголовки — тонкий Manrope с курсивным акцентом |
| v2 | [`/v2/`](https://timurthejohn-droid.github.io/iceberg-website/v2/) | Винтажная эстетика, статичный первый экран |
| v1 | [`/`](https://timurthejohn-droid.github.io/iceberg-website/) | Первая итерация |

## Как устроен первый экран v3

Ткань анимируется **секвенцией из 75 кадров `.webp`**, отрисованных на canvas
(`v3/hero-assets/seq-wide/`, 1,8 МБ) — не видео. Причина: перемотка H.264 требует
декодирования от ближайшего ключевого кадра, из-за чего привязанное к скроллу видео
дёргается. У секвенции операции перемотки нет вовсе.

Кадры догружаются, пока посетитель вводит пароль, — к моменту входа анимация готова.
Прелоадер не исчезает, пока загрузка не закончится.

Разделы: первый экран, «Наши компании» (scrollytelling), группа в цифрах,
клиенты (бегущая строка), «Для кого», контакты с формой заявки.

Шрифты, логотип и графика встроены в HTML — внешних зависимостей нет.
